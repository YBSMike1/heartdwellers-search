Hello!  This Web app is for the sole purpose of finding, in all the past messages released, exactly what Jesus said about the word you search for. For example, if you search for pride, this program will search every message ever released by Mother Clare for any paragraphs in Italic, (Jesus words are always in italic) that used the word "pride". This allows you to dig deeply into only the words spoken by Jesus to Mother Clare, about the word you search for in every message ever realsed quickly! This is similar to our Rhema books that show the same about sins, but this covers every single messages posted. This makes short work to find every message about a specific topic you choose to search for. I have been working on a private version of this for some time and recently found out about the newer options to make this into a functional web app. This will be on our website soon!
