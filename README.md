Hello!  This Web app is for the sole purpose of finding, in all the past messages exactly what Jesus said about the word you search for. For example, if you search for pride, this program will search every message ever 
released by Mother Clare for any paragraphs in Italic, (Jesus words are always in italic) that used the word "pride". This allows you to dig deeply into only the words spoken by Jesus to Mother Clare, about the word you 
search for. This is similar to our Rhema books that show the same about sins. This makes short work to find every message about a specific topic you shoose to search for. I have been working on a private version of this 
for some time and recently found out about the newer options to make this into a web app. Enjoy and God Bless you!  - Mike Fisher
